Uruchomienie - wpisać w terminalu poniższe instrukcje:
npm install express
npm install morgan
node index.js

Strona deweloperska - żeby dodać dane testowe należy wpisać w terminalu:  POPULATE_DB=1 node index.js

Informacja deweloperska - strona jest stworzona z arkuszem stylu css, podkreślam, ponieważ w ostatnim projekcie pojawiła się uwaga o jego braku
