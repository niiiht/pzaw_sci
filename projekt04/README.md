Uruchomienie - wpisać w terminalu poniższe instrukcje:
npm install express
npm install morgan
npm install cookie-parser
npm install express-session
node index.js

Strona deweloperska - żeby dodać dane testowe należy wpisać w terminalu:  POPULATE_DB=1 node index.js
